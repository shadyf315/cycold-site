# CYCOLD

Официальный сайт Minecraft-сервера **CYCOLD**: статус, IP, правила и админ-инструменты.

## Что есть на сайте

- Онлайн-статус сервера.
- Кнопка копирования IP.
- Капча при первом входе на сайт.
- Отдельный экран админки.
- Капча при первом входе в админку.
- Вход по паролю.
- Управление игроками: кик, бан/разбан по нику и IP, бан-листы.

## Структура

- `index.html` — разметка сайта и админ-экрана.
- `styles.css` — стили.
- `script.js` — логика сайта, статуса, капчи и админки.

## Быстрый деплой на GitHub Pages

1. Создай новый репозиторий на GitHub, например `cycold-site`.
2. Загрузи в репозиторий файлы из этой папки `site`.
3. Открой `Settings` -> `Pages`.
4. Включи `Deploy from a branch`.
5. Выбери:
   - Branch: `main`
   - Folder: `/ (root)`
6. Нажми `Save`.

Через 1-3 минуты сайт будет доступен по адресу вида:
`https://<твой-логин>.github.io/cycold-site/`

## Важно по админке

- Для работы действий админки нужен локальный API (`AdminApi`) на `http://127.0.0.1:8787`.
- Если сайт опубликован публично через GitHub Pages, браузер пользователя не сможет обратиться к твоему локальному `127.0.0.1` на другом ПК.
- Поэтому публичную админку лучше:
  - либо отключать;
  - либо переносить API на внешний сервер/VPS и защищать доступ.
